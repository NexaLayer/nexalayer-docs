# Quickstart

This guide shows the typical IPProxy API flow: login, check balance, query products, create an order, and read order details.

## 1. Log in

```bash
curl -X POST "https://api.wanyuagent.com/api/auth/login" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "your_username",
    "password": "your_password"
  }'
```

Save `data.token` from the response.

```bash
export IPPROXY_TOKEN="<token>"
export IPPROXY_BASE="https://api.wanyuagent.com/api"
```

## 2. Check account balance

```bash
curl -X GET "$IPPROXY_BASE/open/app/dashboard/info/v2" \
  -H "Authorization: Bearer $IPPROXY_TOKEN"
```

The response includes current account balance and resource statistics for the authenticated user.

## 3. Query products

```bash
curl -X POST "$IPPROXY_BASE/open/app/product/query/v2" \
  -H "Authorization: Bearer $IPPROXY_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "proxyType": 103
  }'
```

Use the returned `product_no` as `poolId` when creating an order.

## 4. Create a static proxy order

```bash
curl -X POST "$IPPROXY_BASE/open/app/static/order/create/v2" \
  -H "Authorization: Bearer $IPPROXY_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "orderType": "static_proxy",
    "poolId": "ZT_US_New York City_2",
    "userId": 1001,
    "unitPrice": 10,
    "totalAmount": 10,
    "regionCode": "US",
    "countryCode": "US",
    "cityCode": "New York City",
    "staticType": "isp",
    "ipCount": 1,
    "cycleTimes": 1,
    "duration": 30,
    "unit": 1,
    "remark": "API order"
  }'
```

The response returns the created order, including `order_no`. Static proxy instances are available from the static order list or order detail APIs.

## 5. List static orders and proxy instances

```bash
curl -X GET "$IPPROXY_BASE/open/app/static/order/list/v2?page=1&pageSize=10" \
  -H "Authorization: Bearer $IPPROXY_TOKEN"
```

You can search by order number, IP, username, password, or region:

```bash
curl -X GET "$IPPROXY_BASE/open/app/static/order/list/v2?search=192.0.2.10&page=1&pageSize=10" \
  -H "Authorization: Bearer $IPPROXY_TOKEN"
```

## 6. Create a dynamic proxy order

```bash
curl -X POST "$IPPROXY_BASE/open/app/order/create/v2" \
  -H "Authorization: Bearer $IPPROXY_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "orderType": "dynamic_proxy",
    "poolId": "dynamic_ipweb",
    "trafficAmount": 1,
    "userId": 1001,
    "unitPrice": 5,
    "totalAmount": 5,
    "ipCount": 1,
    "duration": 30,
    "unit": 1,
    "remark": "API dynamic order"
  }'
```

## Common next calls

| Task | Endpoint |
| --- | --- |
| Create sub-user | `POST /open/app/user/v2` |
| Query current balance | `GET /open/app/user/balance/current` |
| Query regions | `GET /open/app/product/area/v2` |
| Renew static order | `POST /open/app/static/order/renew` |
| Release static order | `POST /open/app/static/order/{order_no}/cancel` |
| Add IP whitelist | `POST /open/app/proxy/addIpWhiteList/v2` |
| Transaction records | `GET /open/app/user/transactions` |
