# 快速开始

本页演示典型 API 调用流程：登录、检查余额、查询产品、创建订单、查询订单代理信息。

## 1. 登录

```bash
curl -X POST "https://api.wanyuagent.com/api/auth/login" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "your_username",
    "password": "your_password"
  }'
```

保存响应中的 `data.token`。

```bash
export ZTPROXY_TOKEN="<token>"
export ZTPROXY_BASE="https://api.wanyuagent.com/api"
```

## 2. 查询账户余额

```bash
curl -X GET "$ZTPROXY_BASE/open/app/dashboard/info/v2" \
  -H "Authorization: Bearer $ZTPROXY_TOKEN"
```

响应会返回当前账号的余额和资源统计。

## 3. 查询产品

```bash
curl -X POST "$ZTPROXY_BASE/open/app/product/query/v2" \
  -H "Authorization: Bearer $ZTPROXY_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "proxyType": 103
  }'
```

返回结果中的 `productId` 可作为创建订单时的 `poolId`。静态 `103` 产品会返回 `areaCode`、`countryCode`、`stateCode` 和 `cityCode`。

如果需要查询城市名称或 IP 段，可继续调用：

```bash
curl "$ZTPROXY_BASE/open/app/product/area/v2?proxyType=103&productNo=<productId>" \
  -H "Authorization: Bearer $ZTPROXY_TOKEN"

curl "$ZTPROXY_BASE/open/app/product/ip-ranges/v2?proxyType=103&countryCode=US&cityCode=NYC" \
  -H "Authorization: Bearer $ZTPROXY_TOKEN"
```

## 4. 创建静态代理订单

```bash
curl -X POST "$ZTPROXY_BASE/open/app/static/order/create/v2" \
  -H "Authorization: Bearer $ZTPROXY_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "orderType": "static_proxy",
    "poolId": "ZT_US_New York City_2",
    "userId": 1001,
    "unitPrice": 10,
    "totalAmount": 10,
    "regionCode": "US",
    "countryCode": "US",
    "cityCode": "New York City",
    "staticType": "isp",
    "ipCount": 1,
    "cycleTimes": 1,
    "duration": 30,
    "unit": 1,
    "remark": "API order"
  }'
```

响应会返回创建后的订单信息，其中包括 `order_no`。静态代理实例可通过静态订单列表或订单详情接口查询。

## 5. 查询静态订单和代理实例

```bash
curl -X GET "$ZTPROXY_BASE/open/app/static/order/list/v2?page=1&pageSize=10" \
  -H "Authorization: Bearer $ZTPROXY_TOKEN"
```

也可以按订单号、IP、用户名、密码或地区搜索：

```bash
curl -X GET "$ZTPROXY_BASE/open/app/static/order/list/v2?search=192.0.2.10&page=1&pageSize=10" \
  -H "Authorization: Bearer $ZTPROXY_TOKEN"
```

## 6. 创建动态代理订单

```bash
curl -X POST "$ZTPROXY_BASE/open/app/order/create/v2" \
  -H "Authorization: Bearer $ZTPROXY_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "orderType": "dynamic_proxy",
    "poolId": "<dynamic_product_no>",
    "trafficAmount": 1,
    "userId": 1001,
    "unitPrice": 5,
    "totalAmount": 5,
    "ipCount": 1,
    "duration": 30,
    "unit": 1,
    "remark": "API dynamic order"
  }'
```

## 常用后续接口

| 任务 | 接口 |
| --- | --- |
| 创建下级用户 | `POST /open/app/user/v2` |
| 查询当前余额 | `GET /open/app/user/balance/current` |
| 查询地区 | `GET /open/app/product/area/v2` |
| 动态子账户 | 联系 ZTProxy 支持开通对外路径 |
| 动态代理提取 | `POST /business/dynamic-proxy/extract` |
| 查询动态流量 | `GET /user/{user_id}/products-flow` |
| 静态订单续费 | `POST /open/app/static/order/renew` |
| 释放静态订单 | `POST /open/app/static/order/{order_no}/cancel` |
| USDT 充值 | `POST /usdt/create-order` |
| 查询交易记录 | `GET /open/app/user/transactions` |
