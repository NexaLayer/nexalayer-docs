# 介绍

ZTProxy 提供面向合作方、代理商系统和指纹浏览器场景的 HTTP API。你可以通过 API 完成登录、用户管理、产品与地区查询、动态代理订单、静态代理订单、续费、白名单、余额和交易记录查询。

## 基础地址

所有示例默认使用生产 API 域名：

```text
https://api.wanyuagent.com/api
```

如果通过指纹浏览器访问 ZTProxy API，也使用同一个 API 域名：`api.wanyuagent.com`。

## 响应格式

大多数接口返回：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

部分历史接口使用 `msg` 字段：

```json
{
  "code": 0,
  "msg": "success",
  "data": {}
}
```

通常 `code: 0` 表示成功。非 0 `code` 或 HTTP 4xx/5xx 表示请求失败。

## 核心概念

| 概念 | 说明 |
| --- | --- |
| 用户 | ZTProxy 平台账号。代理商可以创建和管理自己的下级用户。 |
| 产品 | 可购买的代理资源包，通过 `product_no` / `poolId` 标识。 |
| 动态订单 | 按流量使用的动态代理订单。 |
| 静态订单 | 固定 IP 订单，包含地区、时长、IP 数量、项目或 CIDR 等配置。 |
| 实例 | 静态订单下的具体代理 IP、端口、账号和密码。 |
| 余额 | 创建订单和续费时扣减的账户可用金额。 |

## 文档范围

- **认证**：`POST /auth/login`
- **用户**：创建下级用户、查询当前余额
- **产品与地区**：查询可用代理产品、库存、地区
- **订单**：创建动态/静态订单、查询静态订单、续费、释放资源
- **代理访问**：静态实例凭证、白名单管理
- **账单记录**：余额、消费记录、充值/调整记录
