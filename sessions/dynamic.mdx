# Dynamic orders

Dynamic proxy orders are traffic-based resources for rotating proxy usage.

## Create dynamic order

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/order/create/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "orderType": "dynamic_proxy",
    "poolId": "dynamic_ipweb",
    "trafficAmount": 1,
    "unitPrice": 5,
    "totalAmount": 5,
    "remark": "API dynamic order",
    "userId": 1001,
    "ipCount": 1,
    "duration": 30,
    "unit": 1
  }'
```

## Fields

| Field | Type | Required | Description |
| --- | --- | :---: | --- |
| `orderType` | string | Yes | Use `dynamic_proxy`. |
| `poolId` | string | Yes | Dynamic product number. |
| `trafficAmount` | number | Yes | Traffic quantity. |
| `userId` | number | Yes | Target user ID. |
| `unitPrice` | number | Yes | Unit price. |
| `totalAmount` | number | Yes | Total amount. |
| `remark` | string | No | Order note. |

## Flow usage

For user-level remaining dynamic flow, use:

```bash
curl -X GET "https://api.wanyuagent.com/api/user/1001/products-flow" \
  -H "Authorization: Bearer <token>"
```

For a specific product:

```bash
curl -X GET "https://api.wanyuagent.com/api/user/1001/remaining-flow/dynamic_ipweb" \
  -H "Authorization: Bearer <token>"
```
