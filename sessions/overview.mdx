# 订单概览

ZTProxy 直接对外提供订单 API。动态订单用于按流量使用的动态代理资源，静态订单用于固定 IP 资源。

## 接口概览

| 方法 | 接口 | 说明 |
| --- | --- | --- |
| `POST` | `/open/app/order/create/v2` | 创建动态代理订单。 |
| `POST` | `/open/app/static/order/create/v2` | 创建静态代理订单。 |
| `GET` | `/open/app/static/order/list/v2` | 查询静态订单和代理实例。 |
| `GET` | `/open/app/static/order/statistics` | 查询静态订单状态统计。 |
| `POST` | `/open/app/static/order/renew` | 静态代理订单或实例续费。 |
| `POST` | `/open/app/static/order/{order_no}/cancel` | 释放或取消静态订单。 |

## 通用订单字段

| 字段 | 说明 |
| --- | --- |
| `orderType` | `dynamic_proxy` 或 `static_proxy`。 |
| `poolId` | 产品查询返回的产品编号。 |
| `userId` | 目标用户 ID。 |
| `unitPrice` | 单价。 |
| `totalAmount` | 订单总金额。 |
| `remark` | 可选备注。 |

## 状态和响应

订单创建成功时返回 `code: 0`，订单对象位于 `data`。常用响应字段包括：

| 字段 | 说明 |
| --- | --- |
| `order_no` | ZTProxy 订单号。 |
| `app_order_no` | 应用订单号。 |
| `status` | 当前订单状态。 |
| `product_no` / `pool_type` | 已购买产品。 |
| `amount` / `total_amount` | 扣费金额。 |
| `created_at`、`updated_at` | 订单时间。 |

固定 IP 资源请阅读 [静态订单](/sessions/static)，按流量使用的资源请阅读 [动态订单](/sessions/dynamic)。
