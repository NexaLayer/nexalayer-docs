# Rotation

IPProxy does not expose a session-style rotate abstraction in the public API. Dynamic proxy rotation is handled by the dynamic proxy product and extraction/access method configured for your account.

For public API integrations, use:

- `POST /open/app/order/create/v2` to create dynamic proxy orders.
- Dynamic product and flow APIs to check available products and usage.
- Product-specific proxy extraction/access instructions provided with your account.

Contact IPProxy support if your integration needs a dedicated dynamic proxy extraction endpoint enabled.
