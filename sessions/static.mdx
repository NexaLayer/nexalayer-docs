# 静态订单

静态代理订单用于购买固定 IP 资源，需要指定产品、地区、时长和 IP 数量。

## 创建静态订单

使用产品查询接口返回的 `productId`、`countryCode` 和 `cityCode` 创建静态订单。

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/static/order/create-standard/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "orderType": "static_proxy",
    "poolId": "ZT_US_New York City_2",
    "unitPrice": 10,
    "totalAmount": 10,
    "remark": "API static order",
    "userId": 1001,
    "countryCode": "US",
    "cityCode": "NEW_YORK",
    "staticType": "isp",
    "ipCount": 1,
    "cycleTimes": 1,
    "duration": 30,
    "unit": 1
  }'
```

如果产品查询响应显示该产品为共享静态（`product_type` / `productType` 为 `2` 或 `shared`，或项目列表非空），需要从 `project_list` / `projects` / `projectList` 中选择项目代码，并在请求体中增加 `projectId`：

```json
{
  "orderType": "static_proxy",
  "poolId": "ZT_US_New York City_2",
  "userId": 1001,
  "countryCode": "US",
  "cityCode": "NEW_YORK",
  "staticType": "isp",
  "ipCount": 1,
  "cycleTimes": 1,
  "duration": 30,
  "unit": 1,
  "unitPrice": 10,
  "totalAmount": 10,
  "projectId": "game"
}
```

标准静态订单创建接口采用异步受理模式。接口通过参数、权限、余额和基础价格校验后，会先返回本系统订单号；供应商开通结果、实例 IP、端口、账号密码和最终状态请后续查询订单列表。

受理成功响应示例：

```json
{
  "code": 0,
  "msg": "订单已受理，正在处理中，请通过订单列表查询最终状态",
  "data": {
    "orderNo": "STA20260611102030abc123",
    "order_no": "STA20260611102030abc123",
    "appOrderNo": "APP20260611102030def456",
    "app_order_no": "APP20260611102030def456",
    "status": "processing",
    "queued": true,
    "async": true
  }
}
```

`orderNo` / `order_no` 为本系统订单号，后续查询、续费或释放时优先使用；`appOrderNo` / `app_order_no` 为本系统提交给供应商侧的应用订单号。接口返回 `queued: true` 表示订单已进入后台处理，不代表供应商已完成开通。`ipipvOrderNo`、实例信息和最终状态以后续 `/open/app/static/order/list/v2` 查询结果为准。

## 时长规则

| `unit` | 含义 | 常见 `duration` |
| --- | --- | --- |
| `1` | 天 | 普通静态产品通常为 `30`；部分静态资源产品支持 `30`、`90`、`365`。 |
| `3` | 月 | `1`、`3`、`6`、`12`；部分静态资源产品支持 `1`、`3`、`12`。 |
| `4` | 年 | `1`。 |

## 可选字段

| 字段 | 说明 |
| --- | --- |
| `cidrBlocks` | CIDR 产品可传 `{ "cidr": "x.x.x.x/yy", "count": 1 }` 数组。 |
| `assignedIp` | 指定单个 IP。 |
| `assignedIpList` | 指定多个 IP。 |
| `projectId` | 共享静态代理产品的项目代码。共享静态必传，独享静态不需要传。 |
| `extBandWidth` | 额外带宽。 |
| `bandWidthPrice` | 带宽价格。 |
| `bridgePrice` | 桥接费用。 |

## 查询静态订单

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/static/order/list/v2?page=1&pageSize=10" \
  -H "Authorization: Bearer <token>"
```

搜索示例：

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/static/order/list/v2?search=STA2026&page=1&pageSize=10" \
  -H "Authorization: Bearer <token>"
```

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/static/order/list/v2?region=New%20York&status=active" \
  -H "Authorization: Bearer <token>"
```

列表响应会返回订单字段；如果订单仍在处理，`status` 通常为 `processing` 且可能暂时没有 `instances`。后台开通完成后，订单会更新为成功/失败等最终状态；如果已有实例，也会包含 `instances`，其中包括代理 IP、端口、用户名、密码、状态和到期时间等信息。
