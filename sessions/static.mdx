# Static orders

Static proxy orders provide fixed IP resources for a selected product, region, duration, and IP count.

## Create static order

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/static/order/create/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "orderType": "static_proxy",
    "poolId": "ZT_US_New York City_2",
    "unitPrice": 10,
    "totalAmount": 10,
    "remark": "API static order",
    "userId": 1001,
    "regionCode": "US",
    "countryCode": "US",
    "cityCode": "New York City",
    "staticType": "isp",
    "ipCount": 1,
    "cycleTimes": 1,
    "duration": 30,
    "unit": 1
  }'
```

## Duration rules

| `unit` | Meaning | Common valid `duration` values |
| --- | --- | --- |
| `1` | Day | `30` for normal static products; some static resource products support `30`, `90`, `365`. |
| `3` | Month | `1`, `3`, `6`, `12`; some static resource products support `1`, `3`, `12`. |
| `4` | Year | `1`. |

## Optional static fields

| Field | Description |
| --- | --- |
| `cidrBlocks` | Array of `{ "cidr": "x.x.x.x/yy", "count": 1 }` for CIDR-based products. |
| `assignedIp` | Request one specific IP. |
| `assignedIpList` | Request multiple specific IPs. |
| `projectId` | Project code for shared static products. |
| `extBandWidth` | Additional bandwidth. |
| `bandWidthPrice` | Bandwidth price. |
| `bridgePrice` | Bridge fee. |

## List static orders

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/static/order/list/v2?page=1&pageSize=10" \
  -H "Authorization: Bearer <token>"
```

Search examples:

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/static/order/list/v2?search=STA2026&page=1&pageSize=10" \
  -H "Authorization: Bearer <token>"
```

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/static/order/list/v2?region=New%20York&status=active" \
  -H "Authorization: Bearer <token>"
```

The list response includes order fields and, where available, `instances` containing proxy IP, port, username, password, status, and expiration data.
