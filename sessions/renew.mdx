# Renew orders

Use the static renewal endpoint to extend a static proxy order or instance.

## Endpoint

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/static/order/renew" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "order_no": "STA202605260001abc123",
    "duration": 30,
    "unit": 1,
    "cycleTimes": 1
  }'
```

The renewal payload can vary by product and order type. Include the target order number or instance identifiers, duration fields, and pricing fields required by the product being renewed.

## Before renewing

1. Query the order with `/open/app/static/order/list/v2`.
2. Confirm the current user has permission to renew the target order.
3. Confirm balance with `/open/app/dashboard/info/v2` or `/open/app/user/balance/current`.
4. Use the same duration rules as static order creation.

If balance is insufficient or the order cannot be renewed, the API returns a non-zero `code` or an HTTP error response.
