# 交易记录

ZTProxy 常用交易记录接口分为两类：消费记录和代理额度记录。

## 消费记录

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/user/transactions?page=1&page_size=10&order_type=static" \
  -H "Authorization: Bearer <token>"
```

查询参数：

| 参数 | 说明 |
| --- | --- |
| `page` | 页码。 |
| `page_size` | 每页数量，最大 `100`。 |
| `type` | 交易类型。 |
| `order_type` | `static` 或 `dynamic`。 |
| `start_date`、`end_date` | 日期范围，格式 `YYYY-MM-DD`。 |
| `username` | 用户名筛选，管理员范围。 |
| `order_no` | 订单号筛选。 |
| `keyword` | 搜索备注、类型、订单号或用户名。 |
| `target_user_id` | 目标用户 ID。 |
| `agent_id` | 代理商筛选，管理员范围。 |

## 充值和调整记录

```bash
curl -X GET "https://api.wanyuagent.com/api/transaction/agent/transactions?page=1&page_size=10" \
  -H "Authorization: Bearer <token>"
```

查询参数：

| 参数 | 说明 |
| --- | --- |
| `order_no` | 订单号搜索。 |
| `start_date`、`end_date` | 日期范围，格式 `YYYY-MM-DD`。 |
| `page`、`page_size` | 分页参数。 |
| `user_id` | 可选用户筛选，受账号权限限制。 |

代理商可以查看自己和下级用户记录；普通用户只能查看自己的记录；管理员可以查看权限范围内的全部记录。
