# Transactions

IPProxy exposes two commonly used transaction record APIs: consumption records and agent balance records.

## Consumption records

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/user/transactions?page=1&page_size=10&order_type=static" \
  -H "Authorization: Bearer <token>"
```

Query parameters:

| Parameter | Description |
| --- | --- |
| `page` | Page number. |
| `page_size` | Page size, max `100`. |
| `type` | Transaction type. |
| `order_type` | `static` or `dynamic`. |
| `start_date`, `end_date` | Date range in `YYYY-MM-DD` format. |
| `username` | Username filter, admin scope. |
| `order_no` | Order number filter. |
| `keyword` | Search remark, type, order number, or username. |
| `target_user_id` | Target user ID. |
| `agent_id` | Agent filter, admin scope. |

## Recharge and adjustment records

```bash
curl -X GET "https://api.wanyuagent.com/api/transaction/agent/transactions?page=1&page_size=10" \
  -H "Authorization: Bearer <token>"
```

Query parameters:

| Parameter | Description |
| --- | --- |
| `order_no` | Order number search. |
| `start_date`, `end_date` | Date range in `YYYY-MM-DD` format. |
| `page`, `page_size` | Pagination. |
| `user_id` | Optional user filter, subject to account permissions. |

Agents can view their own and sub-user records. Normal users can only view their own records. Administrators can view all permitted records.
