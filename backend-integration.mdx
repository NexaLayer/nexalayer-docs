# Backend integration

External systems should integrate with IPProxy through the public API domain only:

```text
https://api.wanyuagent.com/api
```

Do not call upstream supplier APIs directly unless IPProxy support explicitly provides a separate integration.

## Recommended integration pattern

1. Keep IPProxy credentials in your backend service.
2. Login and cache the Bearer token.
3. Re-login on `401`.
4. Query products and balance before creating orders.
5. Persist `order_no`, `app_order_no`, `product_no`, target `userId`, request payload, and response body.
6. Use static order list and transaction APIs for reconciliation.

## Fingerprint browser integrations

When using the fingerprint browser to access IPProxy APIs, use the same domain:

```text
api.wanyuagent.com
```

Prefer performing privileged API calls from your backend rather than exposing credentials inside browser scripts.
