# 后端集成

外部系统应通过 ZTProxy 对外 API 域名进行集成：

```text
https://api.wanyuagent.com/api
```

除非 ZTProxy 支持团队明确提供独立集成方式，否则不要直接调用上游供应商 API。

## 推荐集成方式

1. 在你的后端服务中保存 ZTProxy 账号凭证。
2. 登录并缓存 Bearer Token。
3. 遇到 `401` 时重新登录。
4. 创建订单前查询产品和余额。
5. 持久化 `order_no`、`app_order_no`、`product_no`、目标 `userId`、请求参数和响应体。
6. 使用静态订单列表和交易记录接口做对账。

## 指纹浏览器集成

通过指纹浏览器访问 ZTProxy API 时，使用同一域名：

```text
api.wanyuagent.com
```

建议将带权限的 API 调用放在你的后端完成，不要把账号密码或 token 暴露在浏览器脚本中。
