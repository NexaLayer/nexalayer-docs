# Stats

Use dashboard and order-statistics endpoints for account and resource summaries.

## Dashboard summary

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/dashboard/info/v2" \
  -H "Authorization: Bearer <token>"
```

Admins can pass `target_user_id` where permitted.

## Static order statistics

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/static/order/statistics" \
  -H "Authorization: Bearer <token>"
```

The response includes counts such as total, active, processing, pending, expired, cancelled, failed, expiring soon, and cooling-period orders.

## Static resource stats

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/static/resource/stats" \
  -H "Authorization: Bearer <token>"
```

## Dynamic resource stats

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/dynamic/resource/stats?productNo=dynamic_ipweb" \
  -H "Authorization: Bearer <token>"
```
