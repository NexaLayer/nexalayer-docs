# 冒烟测试

正式启用集成前，建议先执行以下检查：

1. `GET https://api.wanyuagent.com/health` 返回健康状态。
2. `POST /api/auth/login` 返回 `code: 0` 和 `data.token`。
3. `GET /api/open/app/dashboard/info/v2` 返回当前账号数据。
4. `POST /api/open/app/product/query-standard/v2` 返回可用产品。
5. 标准产品项包含 `countryCode`、`cityCode` 和 `mappingStatus`。
6. `GET /api/open/app/static/order/list/v2?page=1&pageSize=10` 返回分页数据。
7. `GET /api/open/app/user/transactions?page=1&page_size=10` 返回消费记录。
8. 保护接口缺少 Bearer Token 时返回 `401`。

如果冒烟测试包含创建订单，请选择允许的最小规格产品，并在请求前确认余额。
