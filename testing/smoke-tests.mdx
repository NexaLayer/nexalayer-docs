# Smoke tests

Run these checks before enabling an integration.

1. `GET https://api.wanyuagent.com/health` returns healthy status.
2. `POST /api/auth/login` returns `code: 0` and `data.token`.
3. `GET /api/open/app/dashboard/info/v2` returns current account data.
4. `POST /api/open/app/product/query/v2` returns available products.
5. `GET /api/open/app/product/area/v2` returns supported areas.
6. `GET /api/open/app/static/order/list/v2?page=1&pageSize=10` returns paginated data.
7. `GET /api/open/app/user/transactions?page=1&page_size=10` returns consumption records.
8. Missing Bearer token on a protected endpoint returns `401`.

For order-creation smoke tests, use the smallest permitted product and confirm balance before sending the request.
