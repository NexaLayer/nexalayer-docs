# 端到端测试

完整端到端测试建议覆盖真实集成路径：

1. 健康检查：`GET https://api.wanyuagent.com/health`。
2. 登录：`POST /api/auth/login`。
3. 仪表盘：`GET /api/open/app/dashboard/info/v2`。
4. 产品：`POST /api/open/app/product/query/v2`。
5. 地区：`GET /api/open/app/product/area/v2`。
6. 余额：`GET /api/open/app/user/balance/current`。
7. 创建订单：使用最小可用测试产品创建静态或动态订单。
8. 查询订单：查询静态订单列表或动态订单响应。
9. 交易记录：`GET /api/open/app/user/transactions`。
10. 清理：按需释放静态测试资源。

安全断言：

- 保护接口缺少 token 时返回 `401`。
- 无效 token 会被拒绝。
- 代理商账号不能操作无关用户。
- 无效产品、无效时长或余额不足时返回明确失败响应。
