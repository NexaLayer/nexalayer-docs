# 测试概览

建议按生产集成会调用的真实接口设计测试流程。

## 推荐测试层级

| 层级 | 范围 |
| --- | --- |
| 冒烟测试 | 登录、仪表盘、产品查询、静态订单列表。 |
| 订单流程 | 创建小规格动态或静态订单，然后查询订单状态。 |
| 权限测试 | 确认代理商只能操作自己的下级用户。 |
| 账务测试 | 对比下单前后的余额和交易记录。 |
| 错误处理 | 缺少 token、无效 token、余额不足、产品无效。 |

## 环境

- 基础地址：`https://backend.ztproxy.net/api`
- 健康检查：`https://api.wanyuagent.com/health`
- 使用专用测试账号和测试下级用户。
- 自动化测试中避免创建生产规格订单。

## 清理

静态代理测试结束后，如果产品策略允许，请释放测试资源。
