# 遥测

当前 ZTProxy 对外 API 文档不定义客户端遥测上报接口。集成方可通过订单列表、仪表盘、资源统计和交易记录接口完成运行状态查看和审计。

相关接口：

| 接口 | 用途 |
| --- | --- |
| `GET /open/app/dashboard/info/v2` | 账户和资源概览。 |
| `GET /open/app/static/order/list/v2` | 静态订单和代理实例状态。 |
| `GET /open/app/static/order/statistics` | 静态订单状态统计。 |
| `GET /open/app/user/transactions` | 消费记录。 |

如果需要自定义回调或遥测上报，请先与 ZTProxy 支持确认字段和接入方式。
