# Health

For API-level health, call:

```bash
curl -X GET "https://api.wanyuagent.com/health"
```

For account and resource health, use:

- `GET /api/open/app/dashboard/info/v2`
- `GET /api/open/app/static/order/statistics`
- `GET /api/open/app/static/order/list/v2`

These endpoints provide the practical status information needed for integrations and reconciliation.
