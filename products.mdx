# 产品

产品定义了可购买的代理资源。创建订单前，请先查询产品，确认正确的 `poolId`、地区、时长、IP 数量和价格。

## 查询产品

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/product/query/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "proxyType": 103
  }'
```

常用响应字段：

| 字段 | 说明 |
| --- | --- |
| `product_no` | 产品编号，创建订单时作为 `poolId` 使用。 |
| `product_name` | 产品显示名称。 |
| `proxy_type` | 代理类型，常见值：`103` 静态，`104` 动态。 |
| `inventory` | 可用库存。 |
| `country_code`、`state_code`、`city_code` | 位置字段。 |
| `duration`、`unit` | 支持的时长配置。 |
| `global_price` | 已配置的默认销售价格。 |
| `band_width`、`band_width_price` | 带宽和带宽价格。 |
| `cidr_info` | 支持 CIDR 产品时的网段库存和价格。 |
| `project_list` | 共享静态代理产品的项目库存。 |

## 查询地区

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/product/area/v2?proxyType=103&productNo=ZT_US_New%20York%20City_2" \
  -H "Authorization: Bearer <token>"
```

返回的区域、国家、城市代码可用于静态订单请求中的 `regionCode`、`countryCode` 和 `cityCode`。

## 常见产品类型

| 产品 | 典型用途 | 下单接口 |
| --- | --- | --- |
| 静态代理 | 固定 IP、固定时长、稳定代理账号密码 | `POST /open/app/static/order/create/v2` |
| 动态代理 | 按流量使用的动态代理资源 | `POST /open/app/order/create/v2` |
| 共享静态代理 | 带 `project_list` / 项目库存的静态产品 | `POST /open/app/static/order/create/v2`，并传入 `projectId` |

## 选择产品建议

1. 按 `proxyType` 查询产品。
2. 检查 `inventory` 和地区字段。
3. 确认时长规则。普通静态产品中，`unit` 支持 `1` 天、`3` 月、`4` 年，具体 `duration` 取值以产品配置为准。
4. 创建订单时使用 `product_no` 作为 `poolId`。
5. 如果产品包含 `project_list`，请选择库存充足的 `projectId`。
