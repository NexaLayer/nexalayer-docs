# 产品

产品定义了可购买的代理资源。创建订单前，请先查询产品，确认正确的 `poolId`、地区、时长、IP 数量和价格。

## 查询产品

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/product/query/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "proxyType": 103
  }'
```

如果已经从产品响应中拿到了某个产品的国家或城市代码，也可以用同一套代码继续筛选：

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/product/query/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "proxyType": 103,
    "countryCode": "<countryCode-from-product>",
    "cityCode": "<cityCode-from-product>"
  }'
```

常用响应字段：

| 字段 | 说明 |
| --- | --- |
| `productId` | 产品编号，创建订单时作为 `poolId` 使用。 |
| `name` | 产品显示名称。 |
| `proxyType` | 代理类型，常见值：`103` 静态，`104` 动态。 |
| `stock` | 可用库存。 |
| `area` / `areaCode` | 区域代码。 |
| `country` / `countryCode` | 国家代码。 |
| `state` / `stateCode` | 州、省或行政区代码。 |
| `city` / `cityCode` | 城市代码。 |
| `duration`、`unit` | 支持的时长配置。 |
| `price` | 当前产品默认销售价格。 |

示例响应：

```json
}
```

## 地区代码来源

静态 `103` 产品没有全局统一的国家或城市标准表。请以 `POST /open/app/product/query/v2` 返回的每个产品字段为准：

| 字段 | 用途 |
| --- | --- |
| `area` / `areaCode` | 创建静态订单时的 `regionCode`。 |
| `country` / `countryCode` | 创建静态订单时的 `countryCode`。 |
| `state` / `stateCode` | 产品所在州、省或行政区，仅在返回时使用。 |
| `city` / `cityCode` | 创建静态订单时的 `cityCode`。 |

不同供应商和不同产品可能使用不同的国家、城市代码格式。不要把某个产品返回的 `countryCode` 或 `cityCode` 直接套用到其他产品；下单时应使用同一个 `productId` 响应项里的地区字段。

## 查询国家或城市拥有的 IP 段

使用 `GET /open/app/product/ip-ranges/v2` 查询指定国家、城市或产品下可用的 CIDR/IP 段列表。

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/product/ip-ranges/v2?proxyType=103&countryCode=<countryCode-from-product>&cityCode=<cityCode-from-product>" \
  -H "Authorization: Bearer <token>"
```

查询参数：

| 参数 | 必填 | 说明 |
| --- | :---: | --- |
| `proxyType` | 否 | 产品类型，默认 `103`。 |
| `productNo` | 否 | 按产品编号筛选。 |
| `areaCode` | 否 | 按区域代码筛选。 |
| `countryCode` | 否 | 按国家代码筛选。 |
| `cityCode` | 否 | 按城市代码筛选。 |
| `includeDisabled` | 否 | 是否包含停用 IP 段，默认 `false`。 |

## 常见产品类型

| 产品 | 典型用途 | 下单接口 |
| --- | --- | --- |
| 静态代理 | 固定 IP、固定时长、稳定代理账号密码 | `POST /open/app/static/order/create/v2` |
| 动态代理 | 按流量使用的动态代理资源 | `POST /open/app/order/create/v2` |
| 共享静态代理 | 带 `project_list` / 项目库存的静态产品 | `POST /open/app/static/order/create/v2`，并传入 `projectId` |

## 选择产品建议

1. 按 `proxyType` 查询产品。
2. 检查 `inventory` 和地区字段。
3. 确认时长规则。普通静态产品中，`unit` 支持 `1` 天、`3` 月、`4` 年，具体 `duration` 取值以产品配置为准。
4. 创建订单时使用 `product_no` 作为 `poolId`。
5. 如果产品包含 `project_list`，请选择库存充足的 `projectId`。
