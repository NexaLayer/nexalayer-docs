# 产品

产品定义了可购买的代理资源。创建订单前，请先查询产品，确认正确的 `poolId`、地区、时长、IP 数量和价格。

## 查询产品

创建订单前，先通过产品查询接口获取可购买产品、库存、价格和地区代码。

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/product/query-standard/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "proxyType": 103,
    "language": "zh"
  }'
```

可按国家或城市筛选：

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/product/query-standard/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "proxyType": 103,
    "standardCountryCode": "US",
    "standardCityCode": "NEW_YORK",
    "language": "en"
  }'
```

常用响应字段：

| 字段 | 说明 |
| --- | --- |
| `productId` | 产品编号，创建订单时作为 `poolId` 使用。 |
| `name` | 产品显示名称。 |
| `proxyType` | 代理类型，常见值：`103` 静态，`104` 动态。 |
| `stock` | 可用库存。 |
| `countryCode` | 标准国家代码，使用 ISO2，如 `US`、`JP`。 |
| `countryName` | 按 `language` 返回的国家名称。 |
| `countryNameEn` | 英文国家名称。 |
| `countryNameZh` | 中文国家名称。 |
| `cityCode` | 标准城市代码，如 `NEW_YORK`。 |
| `cityName` | 按 `language` 返回的城市名称。 |
| `cityNameEn` | 英文城市名称。 |
| `cityNameZh` | 中文城市名称。 |
| `nameLanguage` | 当前 `countryName` / `cityName` 使用的语言：`en` 或 `zh`。 |
| `duration`、`unit` | 支持的时长配置。 |
| `price` | 当前产品默认销售价格。 |

`language` 可传 `zh` 或 `en`，默认 `en`。当前可购买产品的国家和城市会同时返回 `countryNameEn` / `countryNameZh`、`cityNameEn` / `cityNameZh`，`countryName` 和 `cityName` 会按请求语言返回。

```json
      }

  }
}
```

## 地区代码规则

下单时请使用产品查询接口返回的 `countryCode` 和 `cityCode`：

| 字段 | 用途 |
| --- | --- |
| `countryCode` | 创建标准静态订单时的 `countryCode`。 |
| `cityCode` | 创建标准静态订单时的 `cityCode`。 |

同一个国家或城市可能有多个可购买产品。创建订单时请同时传入对应产品的 `productId`、`countryCode` 和 `cityCode`。

## 查询国家或城市拥有的 IP 段

使用 `GET /open/app/product/ip-ranges/v2` 查询指定产品下可用的 CIDR/IP 段列表。

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/product/ip-ranges/v2?proxyType=103&productNo=<productId>" \
  -H "Authorization: Bearer <token>"
```

查询参数：

| 参数 | 必填 | 说明 |
| --- | :---: | --- |
| `proxyType` | 否 | 产品类型，默认 `103`。 |
| `productNo` | 否 | 按产品编号筛选。 |
| `areaCode` | 否 | 按区域代码筛选。 |
| `countryCode` | 否 | 按国家代码筛选。 |
| `cityCode` | 否 | 按城市代码筛选。 |
| `includeDisabled` | 否 | 是否包含停用 IP 段，默认 `false`。 |
| `language` | 否 | 名称语言，支持 `zh` 或 `en`，默认 `en`。 |

## 常见产品类型

| 产品 | 典型用途 | 下单接口 |
| --- | --- | --- |
| 静态代理 | 固定 IP、固定时长、稳定代理账号密码 | `POST /open/app/static/order/create-standard/v2` |
| 动态代理 | 按流量使用的动态代理资源 | `POST /open/app/order/create/v2` |
| 共享静态代理 | 带 `project_list` / 项目库存的静态产品 | `POST /open/app/static/order/create-standard/v2`，并传入 `projectId` |

## 选择产品建议

1. 按 `proxyType` 查询产品。
2. 检查 `inventory` 和地区字段。
3. 确认时长规则。普通静态产品中，`unit` 支持 `1` 天、`3` 月、`4` 年，具体 `duration` 取值以产品配置为准。
4. 创建订单时使用 `product_no` 作为 `poolId`。
5. 如果产品包含 `project_list`，请选择库存充足的 `projectId`。
