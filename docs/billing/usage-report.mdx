---
title: Usage report
description: Session and account usage.
---

Session-level usage:

```
GET /sessions/{session_id}/usage
```

Returns traffic (e.g. bytes or GB) and optionally cost for that session.

Account-level stats:

```
GET /stats/overview
```

Returns aggregate usage and cost across sessions. See [Stats overview](/stats/overview) and [Traffic cost](/stats/traffic-cost).
