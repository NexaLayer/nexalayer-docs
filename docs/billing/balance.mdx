# Balance

Query current balance before creating sessions or recharging.

## Request

<CodeGroup>

```bash Request
curl -X GET "https://agent-layer.up.railway.app/v1/billing/balance" \
  -H "X-API-Key: <api_key>"
```

```json Response
{
  "success": true,
  "data": {
    "balance": 1000.00,
    "currency": "CNY",
    "frozen": 0.00,
    "available": 1000.00,
    "last_recharge_at": "2026-02-14T08:00:00Z"
  }
}
```

</CodeGroup>

Insufficient balance will result in `INSUFFICIENT_BALANCE` (402) when creating sessions or renewing. Use [Recharge](/docs/docs/billing/recharge) to add funds.
