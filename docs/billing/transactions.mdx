# Transactions

List account transactions (recharges, consumption, etc.) with pagination.

## Request

<CodeGroup>

```bash Request
curl -X GET "https://agent-layer.up.railway.app/v1/billing/transactions?page=1&page_size=20&type=all" \
  -H "X-API-Key: <api_key>"
```

```json Response
{
  "success": true,
  "data": {
    "items": [
      {
        "transaction_no": "T20260215100000xxxx",
        "type": "recharge",
        "amount": 500.00,
        "balance": 1500.00,
        "status": "completed",
        "created_at": "2026-02-15T10:15:00Z"
      },
      {
        "transaction_no": "T20260215100100yyyy",
        "type": "consumption",
        "amount": -25.00,
        "balance": 1475.00,
        "details": { "session_id": "sess_xxx" },
        "created_at": "2026-02-15T10:20:00Z"
      }
    ],
    "pagination": {
      "page": 1,
      "page_size": 20,
      "total": 42,
      "total_pages": 3
    }
  }
}
```

</CodeGroup>

Use `type` to filter (e.g. `recharge`, `consumption`, `all`). Response includes `pagination` for page, page_size, total, total_pages.
