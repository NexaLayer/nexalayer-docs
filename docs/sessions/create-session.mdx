---
title: Create session
description: Create a dynamic or static proxy session.
---

## Request

```
POST /sessions
```

**Headers:** `X-API-Key` or `Authorization: Bearer <token>`

**Body:**

```json
{
  "type": "dynamic",
  "config": {
    "product_no": "out_dynamic_1",
    "traffic_gb": 1,
    "protocol": "http",
    "country": "US"
  }
}
```

For static sessions, `config` includes product and duration (see product docs).

## Response

```json
{
  "data": {
    "session_id": "sess_abc123",
    "status": "active",
    "proxy_config": {
      "host": "proxy.nexalayer.com",
      "port": 8080,
      "username": "...",
      "password": "...",
      "protocol": "http"
    }
  }
}
```

Use `proxy_config` in your HTTP client to route traffic through the session.
