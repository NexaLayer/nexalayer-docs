---
title: Stats overview
description: Account-level statistics.
---

## Request

```
GET /stats/overview
```

**Headers:** `X-API-Key` or `Authorization: Bearer <token>`

## Response

```json
{
  "data": {
    "total_traffic_gb": 12.5,
    "total_sessions": 42,
    "total_cost": 25.00,
    "currency": "USD",
    "period": "month"
  }
}
```

Exact fields may vary; see API reference. Use for dashboards and monitoring.
