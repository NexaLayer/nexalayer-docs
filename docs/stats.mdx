# Stats

Aggregate statistics for balance, sessions, traffic, and cost.

## Overview

<CodeGroup>

```bash Request
curl -X GET "https://agent-layer.up.railway.app/v1/stats/overview" \
  -H "X-API-Key: <api_key>"
```

```json Response
{
  "success": true,
  "data": {
    "balance": 1000.00,
    "active_sessions": 5,
    "total_sessions": 20,
    "today_traffic_gb": 2.5,
    "month_traffic_gb": 45.0,
    "today_cost": 25.00,
    "month_cost": 450.00
  }
}
```

</CodeGroup>

## Traffic stats

```bash
curl -X GET "https://agent-layer.up.railway.app/v1/stats/traffic?granularity=day&start_date=2026-02-01&end_date=2026-02-15" \
  -H "X-API-Key: <api_key>"
```

## Cost stats

```bash
curl -X GET "https://agent-layer.up.railway.app/v1/stats/cost?period=month" \
  -H "X-API-Key: <api_key>"
```

## Usage report

```bash
curl -X GET "https://agent-layer.up.railway.app/v1/billing/usage-report?period=week" \
  -H "X-API-Key: <api_key>"
```
