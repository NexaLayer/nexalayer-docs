---
title: Recommend products
description: Get product recommendations based on criteria.
---

## Request

```
POST /products/recommend
```

**Headers:** `X-API-Key` or `Authorization: Bearer <token>`

**Body:**

```json
{
  "type": "dynamic",
  "country": "US",
  "protocol": "http",
  "traffic_gb": 5
}
```

## Response

Returns a list of recommended products (same shape as list products) ordered by fit. Use the returned `product_no` values in `POST /sessions` config.
