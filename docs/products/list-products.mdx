---
title: List products
description: List available proxy products.
---

## Request

```
GET /products
```

**Headers:** `X-API-Key` or `Authorization: Bearer <token>`

## Response

```json
{
  "data": [
    {
      "product_no": "out_dynamic_1",
      "name": "Dynamic HTTP",
      "type": "dynamic",
      "protocols": ["http", "https"],
      "countries": ["US", "GB", ...],
      "pricing": { ... }
    }
  ]
}
```

Use `product_no` in session `config` when creating sessions.
