# Billing

IPProxy's backend CNY balance remains the single source of truth. NexaLayer displays balance, prices, and transactions in USD by converting them with IPProxy's current effective USD/CNY rate. NexaLayer does not maintain a second spendable wallet.

All public USD balances, prices, and charge amounts are rounded to two decimal places. Exchange rates retain four decimal places, while the exact USDT payment amount returned by a recharge order must not be rounded because it identifies the transfer.

Current billing policy:

| Item | Price |
| --- | --- |
| Trial | Contact Telegram support @ztproxy for resource setup |
| Minimum top-up | $10, paid with USDT TRC20 |
| Dynamic Session resources | Live USD conversion from `/products` |
| Static Session resources | Live USD conversion from `/products` |
| Session management | Free for now |

## Pricing

```bash
curl "https://api.nexalayer.net/v1/billing/pricing" \
  -H "X-API-Key: $NEXALAYER_API_KEY"
```

## Balance

```bash
curl "https://api.nexalayer.net/v1/billing/balance" \
  -H "X-API-Key: $NEXALAYER_API_KEY"
```

## Recharge

```bash
curl -X POST "https://api.nexalayer.net/v1/billing/recharge" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $NEXALAYER_API_KEY" \
  -H "Idempotency-Key: recharge-001" \
  -d '{ "amount_usd": 10, "currency": "USDT" }'
```

Recharge requests below $10 are rejected. The response contains the exact `amount_usdt`, `wallet_address`, `network`, and order exchange rate needed to render the TRC20 payment QR code.

## Transactions

```bash
curl "https://api.nexalayer.net/v1/billing/transactions" \
  -H "X-API-Key: $NEXALAYER_API_KEY"
```

## Integration guidance

- read balance before creating paid sessions
- balance checks and resource charges happen only in the backend; NexaLayer does not reserve a second wallet
- send `Idempotency-Key` on write operations
- after support configures a resource, create the Session with `provisioning_mode=existing` to avoid another order or charge
- keep recharge, paid provisioning, and cleanup steps recoverable
- for a trial, contact https://t.me/ZTPROXY (@ztproxy); you can also recharge and use NexaLayer directly
