# Accounts

IPProxy accounts are regular platform users. Administrators and agents can create users through the public API. Agents can create and manage their own sub-users; administrators can manage broader account scopes.

## Create user

```bash
curl -X POST "https://api.wanyuagent.com/api/open/app/user/v2" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "sub_user_001",
    "password": "password123",
    "email": "user@example.com",
    "phone": "13812345678",
    "remark": "Created from API",
    "is_agent": false,
    "balance": 0
  }'
```

## Request fields

| Field | Type | Required | Description |
| --- | --- | :---: | --- |
| `username` | string | Yes | 3-50 characters. Must be unique. |
| `password` | string | Yes | 6-50 characters. |
| `email` | string | No | Email address. |
| `phone` | string | No | 11 digit phone number. |
| `remark` | string | No | Internal note. |
| `is_agent` | boolean | No | Create an agent account. |
| `balance` | number | No | Initial balance. |
| `agent_id` | number | No | Admins can specify the parent agent. |
| `auth_type` | number | No | Agent verification type: `1`, `2`, or `3`. |
| `auth_name` | string | No | Verification name when required. |
| `no` | string | No | Certificate number when required. |

## Current balance

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/user/balance/current" \
  -H "Authorization: Bearer <token>"
```

## Dashboard summary

```bash
curl -X GET "https://api.wanyuagent.com/api/open/app/dashboard/info/v2" \
  -H "Authorization: Bearer <token>"
```

Administrators can pass `target_user_id` to view another user's dashboard summary where permitted.
